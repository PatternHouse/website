About
 










Largest opensource database for all kinds of patterns

PatternHouse is the largest collecion of patterns of all kinds, be it triangular, square, circular, waves, series or whatever we are sure that we have got your needs covered. We offer patterns in all leading languages. The best part of all is that it is opensource meaning anyone is free to copy and modify our code including our website which you are browsing right now.

Find out more





We support five mainstream languages
Whatever language you are comfortable and want to see the sourcecode in that language, we have got that covered. We write C, C++, C#, Java and Python code with possible support for many more coming very soon.




Pattern Recognition Software
Yes! You read that right. If you don't want to manually go through the browse pages and would prefer an automated way to tell what pattern you have? Look no further. Our recognition software contains all the algorithms required to parse the input and match it against its database.
